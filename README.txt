
bot.js handles all commands and references all Repositories

**Commands:**
additem - adds a user-prompted item (that exists on the master item list) to the inventory
avatar - test command, useless
deleteitems - deletes user-prompted item from the inventory
items - reference command, provides lookup info from the master item list
races - reference command, provides lookup info from the race list
roll - dice rolling command
showitems - displays the current items and market value of the inventory
spelllist - reference command, shows the possible spell options for every class
spells - reference command, provides lookup info from the spell list
userinfo - test command, useless


**Repositories:**
items.json - un-editted SRD item list
items2.json - item list being editted into readable form
myitems - dynamic inventory
races - SRD race list
spells - SRD spell list
